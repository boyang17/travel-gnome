# Travel Gnome

Planning your trips start here. Organize places, create itineraries and map your journeys all in one place.  

## Try It Out

<a href="https://travel-gnome.vercel.app/" target="_blank">Launch Travel Gnome</a>

## Features

### Core Functionality

- Store all your trip information without the clutter.
- Add notes to each location and write down tips and recommendations.
- Add all the places you want to go and see them on an interactive map.
- Create lists to help organize locations by day, theme, or priority.
- Drag and drop to reorder your stops, and get a clear overview of your entire trip.
- Turn on routing to see real-time distances and estimated travel times between your locations.
- Visualize your route on the map to see how far apart each place are.
- Reorder stops with a simple drag and drop to optimize your path and make the most of your travel time.

### User Authentication
- Secure sign up and login
- User-specific trip data and session management

### User Experience

- Clean and responsive UI
- Fast and dynamic updates with React
- Real-time database syncing

## Technology

- **React TypeScript**
- **Tailwind CSS**
- **Supabase**
- **PostgreSQL**
